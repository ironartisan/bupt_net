# BUPT Net Login script

This is a simple login script for linux system.



## Installation

```sh
git clone git@github.com:ironartisan/bupt_net.git
vim bupt_net.sh
```

Change `USERNAME` and `PASSWORD` to your account.  

## Getting Started
Usage:
- login
  - login edu-network
- logout
  - logout edu-network
- check
  - check login status 

```sh
chmod +x ./bupt_net.sh

# login
./bupt_net.sh login


# logout
./bupt_net.sh logout

Logout Sucess

# check
./bupt_net.sh check
```
